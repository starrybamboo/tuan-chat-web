# Blocksuite 记录目录

这里放的是阶段性治理记录、边界调整记录和架构对比记录。

和 `architecture/`、`editor/`、`frame/` 的区别：

- `architecture/`：描述当前稳定目录边界
- `editor/` / `frame/`：描述当前稳定专题结构
- `records/`：记录某一轮治理、迁移、对比和收缩成果
