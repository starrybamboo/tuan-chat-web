# 掷骰命令一致性审查 Case

## 审查目标

确认同一条掷骰命令在桌面端和手机端上的含义、结果、可见范围一致。两端表现一致的 case 只写一份统一预期；只有表现不同或能力缺失时，才单独列入“差异/缺失项”。

## 公共命令

这些命令在各规则房间都可使用；其中 `.r` / `.rd` 走公共掷骰逻辑。

| 命令 | 用户意图 | 示例 | 统一预期 | 不一样的地方 |
| --- | --- | --- | --- | --- |
| `.r` | 掷一个骰子表达式；没有参数时使用房间默认骰 | `.r 1d6` -> `掷骰结果：1d6 = 1d6[4] = 4` | 聊天里显示掷骰结果 | 无 |
| `.r 变量` | 使用角色卡里保存的掷骰公式 | 先 `.st 手枪 1d8+1d4`，再 `.r 手枪` -> `手枪 = 1d8+1d4 = ...` | 按角色卡里的“手枪”公式掷骰并显示结果 | 无 |
| `.r @角色 变量` | 帮指定角色按他的角色卡公式掷骰 | A 角色有 `手枪=1d8+1d4`，对 A 执行 `.r 手枪` | 使用 A 角色卡里的“手枪”公式 | 无 |
| `.r N# 表达式` | 重复普通掷骰 | `.r 3# 1d6` -> `掷骰3次:` 后跟 3 行结果 | 聊天里显示 N 次独立结果 | 无 |
| `.r N# 变量` | 用角色卡公式重复掷骰 | 先 `.st 手枪 1d8+1d4`，再 `.r 3# 手枪` | 按“手枪”公式连续掷 N 次 | 无 |
| `.rd` | 快速掷一个指定面数的骰；无参数时默认 D100 | `.rd 20` -> `掷骰结果：1d20 = ...` | 聊天里显示对应面数的结果 | 无 |
| `.ri` | 给角色设置先攻值 | `.ri` | 根据角色敏捷更新先攻 | 无 |
| `.st` | 设置角色卡数值或保存一个掷骰公式；适用于没有自己覆盖 `.st` 的规则 | `.st 手枪 1d4+1d8` | 保存“手枪”公式并提示设置成功 | 无 |
| `.st 属性名 变量/表达式` | 保存一个之后可被 `.r 属性名` 使用的掷骰公式；适用于没有自己覆盖 `.st` 的规则 | `.st 伤害 1d6+db`、`.st 手枪 1d8+1d4` | 之后可用 `.r 伤害` / `.r 手枪` 掷骰 | 无 |
| `.st 属性名 数字` | 设置角色卡上的某个数值；适用于没有自己覆盖 `.st` 的规则 | `.st 力量 70` | 力量改为 70，并显示变更 | 无 |
| `.st 属性名+/-数字` | 增减角色卡上的某个数值；适用于没有自己覆盖 `.st` 的规则 | `.st 力量+10`、`.st 敏捷-5` | 按旧值增减，并显示变更 | 无 |
| `.st show` | 查看角色卡，可只看指定字段 | `.st show 侦查` | 展示指定角色卡内容 | 桌面端打开角色卡窗口；手机端打开手机端角色卡展示 |
| `.setdice` | 设置本房间默认骰面 | `.setdice 20` | 之后无参数 `.r` 使用 D20 | 无 |

## COC 命令

| 命令 | 用户意图 | 示例 | 统一预期 | 不一样的地方 |
| --- | --- | --- | --- | --- |
| `.rc` / `.ra` | 做 COC 技能/属性检定；可加调整值、奖惩骰、重复次数 | `.ra 3# 侦查` | 按当前房规显示 3 次侦查检定 | 无 |
| `.rcb` / `.rab` | 做奖励骰检定 | `.rab 侦查 2` | 显示奖励骰检定结果 | 无 |
| `.rcp` / `.rap` | 做惩罚骰检定 | `.rap 侦查 2` | 显示惩罚骰检定结果 | 无 |
| `.rch` / `.rah` | 做暗骰检定 | `.rah 侦查` | 只有主持人和发送者能看到结果 | 无 |
| `.en` | 做成长检定 | `.en 侦查` | 显示成长检定结果，必要时更新角色卡 | 无 |
| `.sc` | 做理智检定并扣 SAN | `.sc 1/1d6` | 显示理智损失并更新 SAN | 无 |
| `.ti` | 抽一个即时疯狂症状 | `.ti` | 显示症状与持续轮数 | 无 |
| `.li` | 抽一个总结疯狂症状 | `.li` | 显示症状与略过时间 | 无 |
| `.st` | 设置 COC 角色卡字段，支持 `str` 等字段别名 | `.st str 70` | 把力量设置为 70 | 无 |
| `.st 属性名 变量/表达式` | 保存 COC 常用掷骰公式，支持 COC 字段别名 | `.st 手枪 1d8+1d4`、`.st 伤害 1d6+db` | 之后可用 `.r 手枪` / `.r 伤害` 掷骰 | 无 |
| `.st show` | 查看 COC 角色卡 | `.st show 侦查` | 展示指定角色卡内容 | 桌面端打开角色卡窗口；手机端打开手机端角色卡展示 |
| `.setcoc` | 设置 COC 房规 | `.setcoc 2` | 之后检定按 2 号房规判定 | 无 |
