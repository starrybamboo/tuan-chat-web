# 导演控制台设置 WebGAL 空间变量 - Why

## 背景

当前前端支持通过聊天输入 `/var set <key>=<expr>` 发送 `WEBGAL_VAR(11)` 结构化变量消息，并写入 `space.extra.webgalVars`。

## 问题

- 让用户直接在输入框使用 `/var ...` 容易误触、难发现、也不利于“导演工具”统一管理演出/控制指令。

## 目标

- 在 WebGAL 导演控制台中提供“设置变量”入口（按钮/弹窗），由 UI 结构化收集 `key/expr` 后发送 `WEBGAL_VAR(11)`。
- 禁用输入框 `/var ...` 触发变量消息（提示用户使用导演控制台）。
- 保持现有变量持久化与展示逻辑不变：仍写入 `space.extra.webgalVars`，聊天气泡仍以“变量”样式显示。
