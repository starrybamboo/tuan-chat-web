- Git rerere
- Git 冲突自动策略（theirs）

### 实现要点
- 在 `tuan-chat-web` 和 `TuanChat` 设置 `rerere.enabled=true`
- 对冲突文件统一应用 `theirs` 版本并 `git add`
- 继续合并并生成合并提交
- 同步更新知识库与变更记录
