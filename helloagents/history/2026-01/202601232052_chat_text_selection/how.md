# 技术设计: 聊天气泡文本选区保持

## 技术方案
### 核心技术
- React 事件处理
- Selection API

### 实现要点
- 在 `app/components/chat/chatFrame.tsx` 的消息容器 `onClick` 中检测 `window.getSelection()`。
- 当选区存在且包含当前点击目标时，直接返回以保留选区。
- 不影响 `isSelecting`/`ctrlKey` 的多选逻辑与 WebGAL 跳转逻辑。
