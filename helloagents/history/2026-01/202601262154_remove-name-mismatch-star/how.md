# 技术设计: 移除聊天发言人名不一致星号提示

## 技术方案

### 核心技术
- React + TypeScript（tuan-chat-web 前端）

### 实现要点
- 在消息气泡组件中移除 `customRoleName` 存在时额外渲染的 `*` 节点。
- 保留 `displayRoleName = customRoleName || role?.roleName` 的展示逻辑，避免影响导入消息“保留发言人名”的能力。
- 同步更新知识库与变更记录。

## 安全与性能
- **安全:** 仅 UI 展示变更，不涉及权限与数据写入路径调整。
- **性能:** 移除一个条件渲染节点，无负面影响。

## 测试与部署
- **测试:** 运行 `pnpm typecheck`（当前 `dev-jxc` 基线存在既有类型错误，与本次变更无关）。
- **部署:** 常规前端发布流程即可。
