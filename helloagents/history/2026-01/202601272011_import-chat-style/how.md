# 技术设计: 导入对话弹窗去嵌套

## 技术方案
### 核心技术
- React + TypeScript
- Tailwind CSS + daisyUI

### 实现要点
- 调整 `ImportChatMessagesWindow` 根容器为 `w-full h-full min-h-0`，移除内部卡片的固定尺寸/阴影/背景。
- 保持 header/body/footer 的 flex 布局，确保中间区域在单层容器内正常滚动。

## 安全与性能
- **安全:** 仅样式调整，无权限/数据影响。
- **性能:** 不新增渲染负担，维持现有结构。

## 测试与部署
- **测试:** 本地打开导入弹窗，验证长内容/角色列表可滚动到底部且无双层滚动。
- **部署:** 随常规前端发布流程。
